# Payload CMS 学习笔记

本项目是学习 [Payload CMS](https://github.com/payloadcms/payload) 的记录。

## 简介

Payload 是一个基于 Node.js 和 TypeScript 的 Headless CMS 与 Web 应用框架。通过本项目的学习，记录以下内容：

- Payload CMS 的安装与配置
- 集合（Collections）与全局数据（Globals）的定义
- 字段类型与访问控制
- 前后端集成实践
